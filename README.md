# Thermal Runaway Prediction

This project focuses on predicting thermal runaway behavior using data-driven analysis.

## Overview
The notebook analyzes temperature trends and identifies conditions that may lead to thermal runaway.

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Outcomes
- Data preprocessing and visualization
- Identification of critical temperature patterns
- Predictive insights into thermal runaway behavior
